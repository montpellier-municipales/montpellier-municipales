## Pourquoi ce site ?

Les élections municipales sont un moment clé pour notre ville. Pourtant, il est parfois difficile de s'y retrouver entre les multiples listes, les programmes touffus et les promesses de campagne.

C'est pourquoi nous avons créé cette plateforme avec trois objectifs :
1.  **Comparer** facilement les propositions.
2.  **Informer** de manière neutre et factuelle.
3.  **Inclure** tous les montpelliérains grâce au multilinguisme.

## Une démarche Open Source

Ce projet est développé de manière transparente. Le code est ouvert, et nous invitons chacun à contribuer, que ce soit pour corriger des bugs, proposer des traductions ou enrichir le contenu.

Restez connectés pour découvrir les analyses des programmes dès leur publication !
